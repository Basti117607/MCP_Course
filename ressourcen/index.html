Häufig gestellte Fragen

Wie kann ich Zertifikate herunterladen?

Nach Abschluss aller Module können Sie Ihr Zertifikat im Bereich "Mein Profil" herunterladen. Klicken Sie dazu auf "Zertifikate" und dann auf "Herunterladen" neben dem entsprechenden Kurs.

Wie lange habe ich Zugriff auf die Kursinhalte?

Sie haben unbegrenzten Zugriff auf alle Kursinhalte nach dem Kauf. Updates und neue Inhalte werden automatisch verfügbar gemacht.

Gibt es zusätzlichen Support bei Fragen?

Ja, Sie können über das Chat-Widget rechts unten jederzeit Fragen stellen. Unser Supportteam antwortet in der Regel innerhalb von 24 Stunden. Für komplexere Fragen können Sie auch eine E-Mail an support@example.com senden.
